# House-Price-Prediction
A machine learning project predicts house prices based on various property features like location, size, number of rooms, and other housing attributes. This repository covers data cleaning, exploratory data analysis (EDA), feature engineering, model training, evaluation, and price prediction using Python and popular data science libraries.

Features of the project:

 Data cleaning and preprocessing     
 Exploratory Data Analysis (EDA)     
 Feature engineering and selection    
 Machine learning model training    
 Model evaluation and performance comparison   
 House price prediction on new data    

Methods used:

Python programming language        
Pandas, NumPy and Matplotlib libraries

Goal of the project: 

Build a reliable model to estimate real estate prices and understand the factors that influence housing values.
